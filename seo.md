# SEO-план продвижения FORME в топ Google/Яндекс

## Текущая ситуация
- Сайт новый — нет траста у поисковиков
- Одностраничник — мало контента для индексации
- Нет локального SEO (Google Business, 2GIS)

---

## ФАЗА 1: Техническое SEO (1-2 недели)

### Что сделать на сайте:

**1. Создать страницы под кластеры запросов:**
```
/uslugi/personalnye-trenirovki
/uslugi/gruppovye-zanyatiya
/uslugi/stretching
/uslugi/detskaya-komnata
/o-nas
/ceny
/raspisanie
/kontakty
/blog
```

**2. Добавить sitemap.xml и robots.txt**

**3. Подключить аналитику:**
- Google Search Console
- Google Analytics 4
- Яндекс.Метрика
- Яндекс.Вебмастер

**Расходы: 0 тг** (сделаем сами)

---

## ФАЗА 2: Локальное SEO (критически важно!)

### Google Business Profile
- Создать и верифицировать карточку
- Добавить 20+ фото зала
- Указать часы работы, телефон, сайт
- Собирать отзывы (минимум 20-30)

### Яндекс.Справочник
- То же самое для Яндекса

### 2GIS
- Обязательно для Алматы — многие ищут там

### Другие каталоги:
- Флампа
- Zoon
- Relax.kz
- Chocotravel (раздел фитнес)

**Расходы: 0-50,000 тг** (2GIS премиум опционально)

## ФАЗА 3: Контентная стратегия (постоянно)

### Блог — главный драйвер трафика

**Статьи под запросы (2-4 в месяц):**

| Запрос | Статья |
|--------|--------|
| упражнения для похудения для девушек | "15 лучших упражнений для похудения в зале" |
| как начать заниматься фитнесом | "Гид для новичков: первый месяц в зале" |
| питание для похудения | "Что есть до и после тренировки" |
| тренировки для ягодиц | "Программа тренировок на ягодицы" |
| женский фитнес алматы | "Как выбрать фитнес-клуб в Алматы" |
| приватный фитнес | "Почему женщины выбирают приватные залы" |

**Формат статей:**
- 1500-3000 слов
- Фото/видео из вашего зала
- Внутренние ссылки на услуги
- CTA на запись

**Расходы: 15,000-30,000 тг/статья** (копирайтер) или 0 если писать самим/через AI

## ФАЗА 4: Ссылочное продвижение

### Белые методы (рекомендую):

**1. Гостевые статьи:**
- Женские порталы Казахстана (nur.kz, sxodim.com)
- Фитнес-блоги
- Городские СМИ Алматы

**2. PR-статьи:**
- Tengrinews, Informburo, Zakon.kz
- Раздел "Партнерский материал"

**3. Крауд-маркетинг:**
- Ответы на форумах, в комментариях
- Отзывы с ссылкой на сайт

**4. Социальные сигналы:**
- Instagram — ссылка в био и сторис
- YouTube (видео тренировок, туры по залу)

### Покупка ссылок (серый метод):

**Риски:** Google может наказать за неестественные ссылки

Если решите покупать:
- Только с тематических сайтов (фитнес, здоровье, женские)
- Не более 5-10 ссылок в месяц
- Анкоры разбавлять (не только "фитнес алматы")
- Биржи: Miralinks, GoGetLinks, SAPE (с осторожностью)

**Расходы:**
- Гостевые статьи: 30,000-100,000 тг/шт
- PR-статьи: 50,000-200,000 тг/шт
- Покупные ссылки: 5,000-20,000 тг/шт

## Инструменты для мониторинга

**Бесплатные:**
- Google Search Console — позиции, клики, ошибки
- Яндекс.Вебмастер — то же для Яндекса
- Google Analytics — трафик, поведение

**Платные (опционально):**
- Serpstat — мониторинг позиций (~$50/мес)
- Ahrefs — анализ ссылок (~$99/мес)
- Keys.so — для Казахстана (~$30/мес)
